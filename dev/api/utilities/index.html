Utilities

Tensor Operations

DynamicalCorrelators.add_single_util_leg — Function
add_single_util_leg(tensor::AbstractTensorMap)

Add a trivial (one-dimensional) utility leg to a tensor map to balance the number of codomain and domain legs. If codomain < domain, adds a leg to the codomain (left); if codomain > domain, adds a leg to the domain (right). This is needed when converting operators to MPO format.

source
DynamicalCorrelators.contract_MPO — Function
contract_MPO(mpo1::FiniteMPO, mpo2::FiniteMPO)

Vertically contract (fuse) two FiniteMPOs into a single FiniteMPO. The boundary tensors are handled separately with explicit fusion, while interior tensors use fuse_mul_mpo. The result is truncated with notrunc().

source
MPSKit.transfer_left — Function
transfer_left(v, A, Ab)

Left transfer operation for a 4-leg boundary tensor v through an MPS tensor A and its conjugate Ab. Used in multi-site correlation function contractions where extra operator legs need to be propagated through the chain via braiding tensors (τ).

Coordinate Utilities

DynamicalCorrelators.cart2polar — Function
cart2polar(point::AbstractArray) -> (r, θ, ϕ)

Convert Cartesian coordinates to polar/spherical coordinates. For 2D points, returns (r, θ, π/2). For 3D points, returns (r, θ, ϕ).

source
DynamicalCorrelators.phase_by_polar — Function
phase_by_polar(theta, phi, phases)

Return a closure that maps a Bond to its phase factor based on its polar angle. Used to assign direction-dependent hopping amplitudes (e.g., d-wave pairing).

Arguments

  • theta: array of arrays of θ values for each phase group.
  • phi: array of arrays of ϕ values for each phase group.
  • phases: the phase factor to assign for each group.
source
DynamicalCorrelators.sort_by_distance — Function
sort_by_distance(latt::CustomLattice, ij)

Given a flat array ij of site indices (first half = left sites, second half = right sites), compute all pairwise distances and return the site pairs sorted by distance.

Returns (is, js, ds) where is[k]-js[k] is the k-th pair and ds[k] is its distance.
